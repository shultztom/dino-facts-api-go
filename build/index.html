Dinosaur API

Welcome to my Dinosaur API

The list of dinosaurs was pulled from this article.

Similar to this version, but written in Go.

Routes

/dinosaurs

  • Returns the entire list of dinosaurs and their description in JSON

/dinosaurs/random

  • Returns a random dinosaur and their description in JSON

/dinosaurs/random/name

  • Returns a random dinosaur name in JSON

/dinosaurs/random/description

  • Returns a random dinosaur description in JSON
